# RabbitMQ

A RabbitMQ client that's just betterer.

## Prereqs & Install

* Node >=9.10.0
* npm >=6.1.0

Please note that the **TypeScript target is ES6**.

```sh
npm install @usefultools/rabbit-mq
```
